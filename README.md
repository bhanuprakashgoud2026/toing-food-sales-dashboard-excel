# Toing Food Sales Dashboard
## Project Overview
This Excel dashboard analyzes sales performance of a food ordering company named Toing.
The dashboard provides insights into:
- Total Sales
- Total Orders
- Total Profit
- Average Order Value
- Quarterly Sales Performance
- Category-wise Sales Analysis
- State-wise Sales Distribution
- Payment Method Performance
- Profit Trends
---
## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
- Map Visualization
---
## Key KPIs
| KPI | Description |
|------|------------|
| Total Sales | Overall revenue generated |
| Total Orders | Total customer orders |
| Total Profit | Profit earned after costs |
| Average Order Value | Average value per order |
---
## Dashboard Features
### Sales Analysis
- Quarterly sales tracking
- Annual sales overview
### Profit Analysis
- Quarterly profit trend analysis
### Category Analysis
- Food category sales comparison

- Geographic sales distribution across India
